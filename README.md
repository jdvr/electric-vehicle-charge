Electric Vehicle Charge
---

A simple tracker for Electric Vehicle Charges.

- Start date
- Duration
- Amount changed (kw)
- Price (total or by kw)

## Run it

```shell
./gradlew run
```
You can use defined DB of [docker-compose.yaml] by running `docker-compose up`. Sample connection string on [.env.dist]
